# 🚨 HAVEN – AI-Based Covert SOS Platform

> A silent shield, a strong voice — enabling discreet emergency communication under surveillance.

---

## 🌟 Overview

HAVEN is an AI-powered covert SOS platform designed to help users send hidden distress signals without alerting others. It uses steganography and AI to ensure secure and discreet communication in critical situations.

---

## 🔥 Key Features

- 🕵️ **Covert SOS Messaging**  
  Hide distress messages inside normal images using LSB steganography  

- 🤖 **AI-Powered Message Generation**  
  Uses LLMs (LLaMA, Gemini) to convert minimal input into meaningful emergency messages  

- 🚨 **Panic Button System**  
  Sends instant alerts with GPS location within seconds  

- ⚡ **Real-Time Processing**  
  End-to-end pipeline with response time under 5 seconds  

- 🧠 **AI Assistant Support**  
  Provides guidance and assistance in critical situations

## ⚙️ How It Works

1. User inputs a short distress message  
2. AI expands it into a detailed message  
3. System generates an image  
4. Message is encoded using steganography  
5. Image can be shared safely  
6. Authority can decode and respond
